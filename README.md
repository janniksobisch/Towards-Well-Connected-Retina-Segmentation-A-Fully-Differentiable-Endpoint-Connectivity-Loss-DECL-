# Towards Well-Connected Retina Segmentation: A Fully Differentiable Endpoint Connectivity Loss (DECL)

This is the official repository for the paper **"Towards Well-Connected Retina Segmentation: A Fully Differentiable Endpoint Connectivity Loss (DECL)"**. 

This repository will host the implementation of the DECL loss function designed specifically for integration within the [nnU-Net](https://github.com/MIC-DKFZ/nnUNet) framework.

## Code Availability

**The code necessary to implement DECL within the nnU-Net architecture is currently undergoing refactoring and will be made publicly available on GitHub upon the final publication of this paper.**

Thank you for your interest in our work!
